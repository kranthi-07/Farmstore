🌾 FarmStore Website

📌 Overview

FarmStore is a responsive web application designed to showcase agricultural products in a clean and user-friendly interface. The project focuses on delivering a modern UI experience with smooth animations and structured product display.


🚀 Features

- Responsive design for mobile, tablet, and desktop
- Modern UI with smooth animations and transitions
- Organized product listing layout
- User-friendly navigation and clean interface
- Optimized performance for better user experience


🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- MongoDb


🔗 Live Demo

📂 Project Structure

- "index.html" – Main entry point
- "styles.css" – Styling and layout
- "script.js" – Interactive functionality
